### 登录
- **URL**: `/login`
- **Method**: `POST`

**请求体(JSON)**:
```json
{
  "username": "exampleUser",
  "password": "examplePassword"
}
```

**成功响应(JSON)**:
```json
{
  "code": "1",
  "message": "success",
  "data": {
    "userID": 1,
    "username": "exampleUser",
    "role": 0,
    "token": "generated_jwt_token_here"
  }
}
```

**失败响应(JSON)**:
```json
{
  "code": "0",
  "message": "用户名或密码错误",
  "data": null
}
```
